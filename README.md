# Scientific Calculator (Python CLI)

A feature-rich scientific calculator built using Python.
Supports arithmetic, trigonometry, logarithms, exponential, factorial, history, 
ANSI colors, and auto-previous-result usage.

## Features
- +, -, *, /, %, **
- sqrt, sin, cos, tan
- log10, ln, exp
- factorial
- history
- colored CLI interface
- error handling

## How to run
```bash
python kalkulator.py
